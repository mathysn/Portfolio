Heros and Villains

Application web pour la gestion d'organisations de super héros

GitHub

Introduction

Cette application web est issue d'un projet universitaire dans lequel nous devions créer un site web pour gérer des super héroes. L'application permet de consulter les différents super héros, de les ajouter à une équipe et de lier les équipes à des organisations. L'application utilise une API fournie par un professeur pour récupérer les données des super héros.

Front-End

La partie front-end de l'application a été réalisée en utilisant le framework Vue.js et la librairie Vuetify. L'utilisation de Vuetify nous a permis de gagner du temps dans le développement de l'application en utilisant des composants déjà existants.

Back-End

La partie back-end de l'application a été réalisée par le professeur encadrant le projet. Nous devions donc nous connecter à l'API fournie pour récupérer les données via des appels HTTP grâce à un service Axios.

Fonctionnalités

L'application propose plusieurs fonctionnalités telles que :

  • Consultation/création d'une organisations
  • Consultation/création d'une carte équipe
  • Consultation/création d'un héros
  • Liaison d'une équipe à une organisation
  • Liaison d'un héros à une équipe
  • Modification d'un héros
