SAE - EZCON

Application web pour la gestion d'une convention de jeux vidéo

GitHub

Introduction

Cette application web est issue d'un projet universitaire dans lequel nous devions créer un site web pour la gestion d'un événement. Ce site web est donc dédié à une convention de jeux vidéo nommée EZCON qui est sensée se dérouler à l'Atraxion d'Andelnans(90) en France.

Front-End

La partie front-end de l'application a été réalisée en utilisant le framework Vue.js. J'ai pu approfondir mes connaissances en développement web en utilisant Vue Router pour la navigation entre les différentes pages, Vuex pour la gestion de l'état de l'application et Axios pour les appels à l'API.

Back-End

La partie back-end de l'application a été réalisée en Node.js avec Express. Nous avons utilisé une base de données PostgreSQL et Sequelize pour la gestion des données. J'ai appris à créer des routes pour les différentes fonctionnalités de l'application et à gérer les requêtes à la base de données. J'ai également découvert l'outil Swagger pour la documentation de l'API et Postman pour tester les différentes requêtes.

Fonctionnalités

L'application propose plusieurs fonctionnalités telles que :

  • Consultation des stands
  • Consultation d'une carte interactive
  • Création d'un compte utilisateur
  • Réservation de billets
  • Consultation du programme
  • Contact des organisateurs

Conclusion

Ce projet m'a permis de développer mes compétences de développement en Vue.js 2, ma capacité à travailler en groupe ainsi que ma capacité à gérer un projet de A à Z. Cette expérience m'a également permis de créer une application web complète avec un aspect front-end et back-end, ce qui était très enrichissant.
